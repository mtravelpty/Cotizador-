Quiero agregar una nueva sección llamada **Plantillas** para crear cotizaciones rápidas reutilizables tipo circuito / multi-destino.
NO reemplazar el flujo actual del cotizador. Debe coexistir con la cotización manual existente.

OBJETIVO:
Poder guardar estructuras frecuentes como:

* Ciudad + Contadora
* Ciudad + Bocas
* Playa + Ciudad
* Circuitos completos
* Programas multi destino

La idea es que una plantilla permita precargar:

* hoteles
* tours
* traslados
* textos personalizados
* títulos de sección
* notas
* orden visual

========================
ESTRUCTURA GENERAL
==================

Agregar nueva sección en sidebar/menu:

* Plantillas

Dentro:

1. Lista de plantillas guardadas
2. Botón “Nueva plantilla”
3. Duplicar plantilla
4. Editar plantilla
5. Eliminar plantilla
6. Usar plantilla

========================
COMPORTAMIENTO
==============

Cuando el usuario presiona “Usar plantilla”:

* se crea automáticamente una nueva cotización
* se cargan hoteles/tours/traslados definidos
* se respetan precios actuales del tarifario
* el usuario luego puede editar normalmente

La plantilla NO debe guardar precios fijos.
Debe guardar:

* referencias
* estructura
* orden
* textos

Los precios SIEMPRE deben venir del tarifario actual.

========================
ESTRUCTURA DE PLANTILLA
=======================

La plantilla debe permitir bloques dinámicos.

Tipos de bloques:

1. TEXTO
   Texto libre editable.

2. HOTEL
   Permite agregar:

* destino
* hotel
* categoría
* notas

3. TOUR
   Permite seleccionar tours existentes.

4. TRASLADO
   Permite seleccionar traslados existentes.

5. TÍTULO
   Ejemplo:
   “Hoteles en Ciudad de Panamá”

6. NOTA
   Texto destacado.

========================
ORDEN DRAG & DROP
=================

Los bloques deben poder:

* mover arriba/abajo
* reordenarse
* duplicarse
* eliminarse
